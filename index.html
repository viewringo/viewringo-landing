About ViewRingo

ViewRingo is a **BI portal platform** designed to integrate various BI tools such as Power BI, Tableau, Qlik, Superset, MicroStrategy, and Longview into a single, unified UI. It provides consistent authentication and authorization systems across all integrated tools.

Our core features include maximizing user convenience by integrating reports from multiple BI tools, offering unified authentication, menu, and authorization management, and providing consistent UX with personalization features like themes, favorites, and start pages. ViewRingo also experimentally supports an AI-based Co-Pilot (NL2SQL analysis assistant).

Unlock the Power of Your Data

Unified BI Tool Integration

Seamlessly integrate and manage leading BI tools including Power BI, Tableau, Qlik, Superset, Longview, and MicroStrategy. Access all your reports and dashboards from a single, intuitive interface, maximizing user convenience.

Centralized Security & Access

Leverage a robust, centralized authentication and authorization system for all integrated BI tools. Simplify user access control and fortify your data security posture with consistent policies.

Comprehensive Management

Efficiently manage users (auto/multi-registration, password management), menus (BI tool report registration UI), and permissions (groups, matrix) across your entire BI ecosystem. Includes monitoring logs for activity tracking.

Cutting-Edge Technology

Built on ASP.NET Core (.NET 8.0) with TypeScript for dynamic frontend services, AdminLTE-based Razor views, and Dapper for high-performance data access, ensuring scalability and reliability. HTML5 based for multi-device support.

Flexible Data Infrastructure

Primary support for SQL Server and PostgreSQL, with planned support for Oracle and MySQL. This offers unparalleled flexibility for your evolving data infrastructure needs.

Advanced Analytics & Insights

Beyond integration, ViewRingo provides tools for deeper analytics, including experimental AI-based Co-Pilot (NL2SQL analysis assistant) for extracting meaningful insights and making data-driven decisions with confidence.
