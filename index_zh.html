关于 ViewRingo

ViewRingo 是一个 **BI 门户平台**，旨在将 Power BI、Tableau、Qlik、Superset、MicroStrategy 和 Longview 等各种 BI 工具整合到一个统一的用户界面中。它为所有集成工具提供一致的身份验证和授权系统。

我们的核心功能包括通过集成多个 BI 工具的报告来最大化用户便利性，提供统一的身份验证、菜单和授权管理，以及提供一致的用户体验和个性化功能，如主题、收藏夹和起始页。ViewRingo 还实验性地支持基于 AI 的 Co-Pilot (NL2SQL 分析助手)。

释放您的数据力量

统一的 BI 工具集成

无缝集成和管理领先的 BI 工具，包括 Power BI、Tableau、Qlik、Superset、Longview 和 MicroStrategy。从单一、直观的界面访问您的所有报告和仪表板，最大限度地提高用户便利性。

集中式安全和访问

为所有集成 BI 工具利用强大且集中式的身份验证和授权系统。简化用户访问控制并以一致的策略增强您的数据安全态势。

全面的管理

高效管理整个 BI 生态系统中的用户（自动/多重注册、密码管理）、菜单（BI 工具报告注册 UI）和权限（组、矩阵）。包括用于活动跟踪的监控日志。

尖端技术

基于 ASP.NET Core (.NET 8.0) 构建，前端服务采用 TypeScript，Razor 视图基于 AdminLTE，数据访问采用 Dapper，确保高性能和可扩展性。基于 HTML5，支持多设备。

灵活的数据基础设施

主要支持 SQL Server 和 PostgreSQL，并计划支持 Oracle 和 MySQL。这为不断发展的数据基础设施需求提供了无与伦比的灵活性。

高级分析和洞察

除了集成，ViewRingo 还提供更深入的分析工具，包括实验性的基于 AI 的 Co-Pilot (NL2SQL 分析助手)，使用户能够提取有意义的洞察并自信地做出数据驱动的决策。
